# Быстрый запуск через GitHub Pages

```
амый быстрый путь — GitHub Pages:

	1.	Зайди на github.com → New repository
	2.	Назови как угодно, сделай Public
	3.	Загрузи файл — переименуй его в index.html
	4.		Settings → Pages → Branch: main → Save
	5.	Через минуту получишь username.github.io/repo — открой в Safari на iPhone
```

Файл добавлен в репозиторий по запросу. Если хотите — могу переименовать файл, добавить перевод на английский или автоматически создать ветку/PR вместо записи прямо в default branch.
